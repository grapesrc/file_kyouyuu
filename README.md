# ファイルを共有できるツール

これは、同じlan内でファイルを共有できるtoolです。

![logo](https://raw.githubusercontent.com/rihitosan/file_kyouyuu/main/logo.png)

## 使い方

① [参照]ボタンを押してファイルを選ぶ 。
②スマホで表示されているQRコードを読み込む
③[共有(ウィンドウを閉じる)]ボタンを押す
④スマホからQRコードのURLにアクセスする

## 注意

携帯などの端末でQRコードを読み取っても、
[共有(ウィンドウを閉じる)]ボタンを押
さないとファイルは共有されません。

ファイルを共有するときは、共有する端末と同じ
ルーターに接続してください。
